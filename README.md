# Vatsal Jha - Portfolio

A modern, responsive portfolio website built with React.

## 🚀 Getting Started

### Prerequisites
- Node.js (v14+)
- npm

### Installation
```bash
git clone https://github.com/Vatsal-Jha256/PortfolioMain.git
cd PortfolioMain
npm install
npm start
```

## 🛠️ Built With
- React
- Framer Motion
- React Icons
- CSS3
